About UI Utils

UI Utils was created by Coderx Gamer and MrBreakNFix in 2022, when they met on an anarchy server, both trying to find a duplication glitch. They worked together to create what is now known as UI Utils.

The mod adds multiple features to Minecraft GUIs, making it useful for debugging plugins and dupe hunting on servers. It is always updated to the latest stable version of Minecraft.
